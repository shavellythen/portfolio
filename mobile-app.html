Mobile Application

University of Miami

UX / UI Design

Project Overview : The purpose of this project was to design a user-friendly and visually appealing mobile application for the University of Miami.

The app aims to engage with various stakeholders, including alumni, friends, parents, and donors. By providing a seamless and intuitive user experience (UX) combined with an attractive user interface (UI), the university strengthens its relationships with these important constituents and fosters a sense of community.

Goals:
Enhance Alumni Engagement
Foster Friend and Parent Engagement
Encourage Donor Involvement

Key Features:

  • User Registration and Profiles: Seamless registration process for different user types (alumni, friends, parents, donors). User profiles with customizable preferences and information.
  • News and Updates: Dynamic news feed with personalized content based on user preferences. Push notifications for important announcements, events, and news updates.
  • Event Management: Event calendar with details, RSVP functionality, and reminders. Ability to browse and register for events, both on-campus and virtual. Integration with the user's personal calendar.
  • Networking and Connections: Alumni directory with search and filter options. In-app messaging and networking capabilities. Groups and communities for specific interests or affiliations.
  • Giving and Donations: Secure donation functionality with various giving options. Progress updates on fundraising campaigns and projects. Recognition and acknowledgement features for donors.
  • Resource Center: Access to university publications, newsletters, and research. Library resources, career services, and mentorship programs. FAQ section and support channels for user assistance.
